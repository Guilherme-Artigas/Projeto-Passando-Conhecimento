Imagens em HTML 5

Sintaxe de imagens em HTML

A Tag HTML usada para incorporar uma imagem em uma página web é <img>, imagens melhoram muito o design e a aparência de uma página. A Tag <img> é vazia quando criada, contendo apenas atributos para serem preenchidos e ela não possui fechamento, os atributos necessários são src (especifica o caminho para a imagem) e o alt (especifica um texto alternativo para a imagem), esse segundo sendo muito importante e valorizado por mecanismos de busca, pois, auxiliam pessoas com dificuldades visuais a navegarem na internet.

O atributo src

Nota: Quando uma página web é carregada, é o navegador que obtém a imagem de um servidor web e a insere na página. Portanto, certifique-se de que a imagem realmente permaneça no local que foi indicado nesse atributo, caso contrário os visitantes receberão um ícone de link quebrado.

O atributo alt

Esse atributo é necessário, pois fornece um texto alternativo para uma imagem, se o usuário por algum motivo não puder visualizá-la (por causa de conexão lenta, ou dificuldades visuais), o atributo alt será exibido na tela ou lido por leitores de tela.
